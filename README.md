# thought-pattern-observer

Visualize thought and decision patterns. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Run

```bash
go run .
```

## Build

```bash
make build
./bin/thought-pattern-observer
```

## Controls

- `j/k`: navigate
- `Enter`: select/details
- `q`: quit
