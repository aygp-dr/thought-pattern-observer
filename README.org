#+TITLE: thought-pattern-observer
#+STARTUP: overview

* Overview

Visualize thought and decision patterns. Terminal UI built with [Bubble Tea](https://github.com/charmbracelet/bubbletea).

* Quick Start

#+begin_src bash
go run .
# or
make build && ./bin/thought-pattern-observer
#+end_src

* Controls

| Key | Action |
|-----+--------|
| j/k | Navigate |
| Enter | Select/details |
| q | Quit |

* Development

| Target | Description |
|--------+-------------|
| make build | Build binary |
| make run | Build and run |
| make test | Run tests |
| make lint | Run go vet |
